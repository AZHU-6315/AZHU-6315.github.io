CSS基础知识学习

css

层叠样式表（Cascading Style Sheets ，缩写为CSS），是一种样式表语言，用来描述HTML文档的呈现（美化内容）。

CSS引入方式

  • 内部样式表

    • CSS代码写在style标签里面

  • 外部样式表：开发常用

    • CSS代码写在单独的CSS文件中(文件扩展名为.css)

    • 在HTML中使用link标签引入

  • 行内样式：配合JavaScript使用

    • css卸载标签的style属性值里面

基本选择器1

  1. 作用：查找标签，设置样式
  2. 基础选择器：标签选择器，类选择器，id选择器，通配符选择器

标签选择器

使用标签名作为选择器，选中同名标签设置相同格式（无法差异化同名标签的样式）

例如：p，h1，div，a，img……

类选择器

作用：查找标签，差异化设置标签的显示效果

注意：

  • 一个类选择器可以给多个标签使用，一个标签不能加多个类名（即不能出现多个class），但是如果想给一个标签使用多个类名，可以在同一个class里空格隔开，class属性值写多个类名
  • 类名自定义，但是不要用纯数字或者中文
  • 开发习惯：类名见名知意，多个单词可以使用-连接，例如：news-hd

使用步骤：

  1. 定义类选择器，格式为”.类名”

  2. 使用类选择器，即在标签内添加”class=”类名””

id选择器

  1. 作用：查找标签，差异化显示标签的显示效果

  2. 使用场景：id选择器一般配合javascript使用，很少直接用来设置CSS样式、

  3. 使用步骤：

    1. 定义id选择器→#id名

    2. 使用id选择器→标签名加id=”id名”

  4. 使用规则：

    • 同一个id选择器在同一个页面中只能使用一次

通配符选择器

  1. 作用：查找页面中所有标签，设置相同样式

  2. 通配符选择器：*，不需要调用，浏览器自动查找页面内所有标签，设置相同样式

  3. 使用场景：在开发初期格式化标签样式，如标签之间的间距等

画盒子

目标：使用合适的选择器画盒子

相关属性：

属性名 作用
width 宽度
height 高度
background-color 背景颜色

文字控制属性

属性名 作用
font-size 字体大小
font-weight 字体粗细
font-style 字体倾斜
line-height 行高
font-family 字体族（样式：宋体，微软雅黑等）
font 字体复合属性
text-indent 文本缩进
text-align 文本对齐
text-decoration 字体修饰线
color 字体颜色

字体大小

  • 属性值为文字尺寸，最常用单位为px
  • 无单位不显示效果
  • 谷歌浏览器默认字体大小为16px

字体粗细

属性值 正常字体 加粗字体
数字 400 700
单词 normal bold

字体倾斜

  1. 作用：清除文字默认倾斜效果
  2. 属性值：正常：normal；倾斜：italic

行高

  1. 属性值：1.数字+px 2.数字（字体大小属性值的倍数）
  2. 行高：上间距+文本高度+下间距
  3. 找行高值：两行文本顶部间距
  4. 垂直居中使用方法：行高属性值等于盒子高度属性值（使文本居于“行”的中间，只适用于单行文字）

字体族

  1. 属性值：字体名

  2. 拓展：font-family属性值可以书写多个字体名，各字体名之间用逗号隔开，执行顺序是从左到右依次查找

  3. 属性最后设置一个字体族名，网页开发建议使用无衬线字体(sans-serif)

字体符合属性

  1. 使用场景：开发初期设置网页文字公共样式
  2. 复合属性是属性的简写方式，一个属性对应多个值的写法，各属性之间用空格隔开
  3. font：是否倾斜 是否加粗 字号/行高 字体（必须按顺序书写）
  4. 注意：字号和字体值必须书写，否则font属性不生效

文本缩进

  1. 属性值：
    • 数字+px
    • 数字+em（推荐：1em=当前标签的字号大小）

文本对齐

  1. 作用：控制内容水平对齐方式

  2. 属性值

    属性值 效果
    left 左对齐
    center 居中对齐
    right 右对齐

  3. 本质：居中的是文字内容，与标签无关，属性要设置给内容的父级

文本修饰线

  1. 属性名：text-decoration

  2. 属性值：

    属性值 效果
    none 无
    underline 下划线
    iine-through 删除线
    overline 上划线

文字颜色

属性值

颜色表示方式 属性值 说明 使用场景
颜色关键词 颜色英文单词 red、green、blue… 学习测试
rgb表示法 rgb(r,g,b) r,g,b表示红绿蓝三原色，取值0-255 了解
rgba表示法 rgba(r,g,b,a) a表示透明度，取值：0-1 开发使用，实现透明呈现
十六进制表示法 #RRGGBB #000000，#ffcc00，简写：#000，#fc0 开发使用（一般从设计稿直接复制）

调试工具

  2. 使用步骤：
    1. 打开调试工具
      • 浏览器窗口任意位置/选中标签→鼠标右键→检查

选择器2–复合选择器

定义：由两个或多个基础选择器，通过不同的方式组合而成。

作用：更准确，更高效的选择目标元素（标签）

后代选择器

后代选择器：选中某元素的后代元素,子元素，孙级元素等都可选中

选择器写法：父选择器 子选择器{CSS属性}，父子选择器之间用空格隔开

子代选择器

子代选择器：选中某元素中的子代元素（最近的子级）

选择器写法：父选择器>子选择器{CSS属性}，父子选择器之间用**>**隔开

并集选择器

并集选择器：选中多组标签设置相同的样式

选择器写法：选择器1，选择器2，…，选择器n{CSS属性}，选择器之间用**，**隔开

交集选择器

交集选择器：选择同时满足多个条件的元素

选择器写法：选择器1选择器2{CSS属性}，选择器之间连写，没有任何符号

注意：如果交集选择器中有标签选择器，标签选择器必须写在最前面。

伪类选择器

伪类选择器：伪类表示元素状态，选中元素的某个状态设置样式

鼠标悬停状态：选择器：hover{CSS属性}

任何标签都可以设置鼠标悬停的状态

伪类-超链接（拓展）

超链接一共有四个状态

一定要按照LVHA的顺序书写

作用 选择器
访问前 ：link
访问后 ：visited
鼠标悬停的状态 ：hover
点击时激活 ：active

CSS特性

CSS特性：化简代码/定位问题，并解决问题

  • 继承性
  • 层叠性
  • 优先级

继承性

  • 继承性：子级默认继承父级的文字控制属性
  • 注意：如果标签自己有样式，则生效自己的样式，不会继承父级样式

层叠性

  • 相同的属性会产生覆盖：后面的CSS属性覆盖前面的CSS属性
  • 不同的属性会产生叠加：不同的CSS属性都生效

优先级

  • 优先级也叫权重，当一个标签用了多个选择器时，基于不同种类的选择器的匹配规则
  • 规则：优先级高的选择器样式优先生效
  • 公式：通配符选择器<标签选择器<类选择器<id选择器<行内样式<!important
  • ！important：提权功能，提高权重，优先级到最高，慎用

优先级-叠加计算规则

  • 叠加计算：如果是复合选择器，需要进行权重叠加计算
  • 公式：行内样式，id选择器个数，类选择器个数，标签选择器个数（每一级之间不存在进位）
  • 规则：
    • 从左到右依次比较选个数，同一级个数多的优先级高，如果个数相同，则向后比较
    • !important权重最高
    • 继承权重最低

Emmet写法

Emmet写法：代码的简写方式，输入缩写VS Code会自动生成对应的代码

  • HTML

  • CSS：大多数简写方式为属性单词的首字母，想要多个属性的话首字母加号连接

背景属性

背景图

网页中，使用背景图实现装饰性的图片效果

属性名：background-image（Emmet：bgi）

属性值：url(背景图URL)

注意：背景图默认是平铺效果，如果图片尺寸小于盒子尺寸则浏览器会自动复制图片进行填充，可以通过后续的属性值来调整

背景图平铺方式

属性名：background-repeat（bgr）

属性值：

属性值 效果
no-repeat 不平铺
repeat 平铺（默认效果）
repeat-x 水平方向平铺
repeat-y 垂直方向平铺

背景图位置

属性名：background-position（bgp）

属性值：水平方向位置 垂直方向位置

  • 关键字

    关键字 位置
    left 左侧
    right 右侧
    center 居中
    top 顶部
    bottom 底部

  • 坐标

    数字+px，正负都可以

    水平方向：正数向右，负数向左

    垂直方向：正数向下，负数向上

提示：

  • 关键字取值方式写法，可以颠倒取值顺序
  • 可以只写一个关键字，另一个方向默认为居中；
  • 数字只写一个值则表示为水平方向，垂直方向为居中

背景缩放

属性名：background-size(bgz)

作用：设置背景图大小

常用属性值：

  • 关键字：
    • cover:等比例缩放背景图片以完全覆盖背景盒子，可能出现背景图片部分看不见的问题
      • 图片完全覆盖盒子
    • contain:等比例缩放背景图片以完全装入背景区，可能背景区部分空白
      • 图的宽高和盒子尺寸相等时停止缩放
  • 百分比：根据盒子尺寸计算图片大小
    • 100%：图片的宽度缩放至跟盒子宽度一样，图片的高度按照图片比例等比缩放
  • 数字+单位（例如：px）

背景图固定

作用：背景不会随着元素的内容滚动

属性名：background-attachment(bga)

属性值：fixed

背景复合属性

属性名：backgronund（bg）

属性值：背景色 背景图 背景图平铺方式 背景图位置/背景图缩放 背景图固定 （空格隔开各个属性，不区分先后顺序）

显示模式

显示模式：标签（元素）的显示方式

作用：布局网页的时候，根据标签的显示模式选择合适的标签摆放内容

  • 块级元素
    • 独占一行
    • 宽度默认为父级的100%
    • 添加宽高属性生效
  • 行内元素
    • 一行可以共存多个
    • 尺寸由内容决定
    • 加宽高不生效
  • 行内块元素
    • 一行共存多个
    • 默认尺寸由内容撑开
    • 加宽高生效

转换显示模式

属性名：display

属性值：

属性值 效果
block 块级
inline-block 行内块
inline 行内
